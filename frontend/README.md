# Frontend Module

This directory contains the React.js frontend for the Lab 27 application.

## Development

To run the frontend independently:

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```

## Production Build

To create a production-ready build:
```bash
npm run build
```

The resulting files will be in the `build/` directory.
